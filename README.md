# My Personal Website
> [!Note] 
> This website is built on [Hugo Academic CV Theme](https://github.com/HugoBlox/theme-academic-cv), but with the rebranding of HugoBlox, all is history.

## Local Building
1. Install [hugo_extended_0.136.5](https://github.com/gohugoio/hugo/releases/tag/v0.136.5). Add `hugo.exe` to your PATH.

2. Clone the repository and run the following command in the root directory of the project:
```bash
hugo server
```
